--- title: Fixing "iertutil.dll was not found" after a Windows XP update layout: default tags: - Tech Support - windows type: link --- Fixing "iertutil.dll was not found" after a Windows XP update An XP machine wasn't working after an update. When we logged in an error popped up saying `iertutil.dll was not found`, and running `explorer` from task manager gave a cascade of the same errors. Following [these instructions][1] I found `iertutil.dll` in `C:\Windows\system32\dllcache` and copied it into `C:\Windows\system32` using Tak Manager's Open dialog. Grooving.
